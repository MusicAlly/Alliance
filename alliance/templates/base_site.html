Sites using the Alliance app may leave this be, or write their own base_site.html in one of the site's TEMPLATE_DIRS directories. If Django finds a base_site.html file in one of the TEMPLATE_DIRS, it will use that file instead of this one. Note that the replacement file should be stored directly in the template directory, not an alliance/ subfolder.
